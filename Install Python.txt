Install Python from python.org/downloads

Run Python:
(Terminal)
$python3.4
$exit()

Install pip (Python module manager):
Copy/paste get-pip.py from http://pip.readthedocs.org/en/latest/installing.html into get-pip.py text file in Downloads folder
(Terminal)
$ cd ~/Downloads #navigate to Downloads folder
$python3.4 get-pip.py

Use pip to install/update IPython + Notebook + Dependencies:
(Terminal)
$pip3.4 install ipython[all]

Run IPython Notebook:
(Terminal)
$ipython notebook
Exit IPython Notebook:
(close browser)
(Terminal)
CTRL-C
exit? y [Enter]
